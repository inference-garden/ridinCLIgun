# ridinCLIgun

A terminal companion that advises but never acts.

## What is it?

ridinCLIgun is a split-pane terminal app: a real shell on the left, an advisory assistant on the right. It helps you work in the terminal more safely and confidently — without taking control away from you.

## Status

Early development (v0.2). macOS only.

## Quick start

```bash
pip install -e ".[all]"
python -m ridincligun
```

## Shortcuts

- `Ctrl+G` → leader key for app actions
- `Ctrl+G, H` → show help
- `Ctrl+Q` → quit
