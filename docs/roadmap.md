# Roadmap

Where ridinCLIgun is heading. Vision, not promises — this is a personal project
with no fixed timeline.

## v0.3 (current) — "This tool is careful"

Settings UI, toast notifications, clipboard safety, secret detection,
AI review with redaction preview, deep script analysis, onboarding flow.

## v0.4 — "This tool is genuinely useful." (Accessibility)

Multi-language support (DE/FR/EN), model selection improvements,
custom provider endpoints for self-hosted LLMs.

## v0.5 — Learning

Review history, learning mode for beginners, command explanations.

## Future

Linux & Windows support, Homebrew distribution, plugin system.
And this is one reason why I started this: a learning game for (my) Kids in advisory pane to learn about terminal use in a playful way :-)
